# Minor-ML-projects

Three short notebook-based exercises covering core ML / data-analysis ideas.
Each subfolder is self-contained with its own dataset and a brief README.

## Contents

| Folder | What it covers |
|---|---|
| [`linear regression/`](./linear%20regression/) | Linear regression on the cost-vs-revenue movie dataset |
| [`gradient descent/`](./gradient%20descent/) | Gradient descent from scratch, with `sympy` for symbolic derivatives and `mpl_toolkits.mplot3d` 3-D plots |
| [`pandas (basics)/`](./pandas%20%28basics%29/) | Pandas fundamentals on the LSD-vs-math-test psychology dataset (1960s study) |

## Setup

```sh
pip install -r requirements.txt
jupyter lab
```

The notebooks are pedagogical — they demonstrate the building blocks rather
than ship a production model.
